# Übungsaufgaben

## Regression

### Lineare Link-Funktion in `glm()` {.no-number}

```{webr}
#| exercise: ex_1
#| setup: true
library(palmerpenguins)
data(penguins)

# Datensatz bereinigen
penguins_cleaned <- penguins %>% 
  rename(
    bill_depth = bill_depth_mm,
    bill_length = bill_length_mm,
    flipper_length = flipper_length_mm,
    body_mass = body_mass_g
  ) %>% 
  drop_na() %>% 
  filter(
    body_mass < quantile(body_mass, .95)
  ) %>%
  mutate(sex = if_else(sex == "female", 1, 0))
```

```{webr}
#| caption: Linearer Link
#| exercise: ex_1
glm(
  formula = sex ~ bill_depth, 
  data = penguins_cleaned,
  family = _____
) 
```

::: {.solution exercise="ex_1"}

#### Solution

Setze `family = gaussian(link = "identity")`.

```{webr}
#| exercise: ex_1
#| solution: true
glm(
  formula = sex ~ bill_depth, 
  data = penguins_cleaned,
  family = gaussian(link = "identity")
) 
```
